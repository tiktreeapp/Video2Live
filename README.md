# Video2Live

从视频一键生成 Live Photo 的 iOS 应用（SwiftUI）。

## 功能
- 首页 UI 与基本交互
- 核心 Video2Live 流程（从视频生成 Live Photo/动图）

## 环境要求
- Xcode 15+
- iOS 16+（按需调整）

## 快速开始
```bash
git clone https://github.com/tiktreeapp/Video2Live.git
cd Video2Live
open Video2Live.xcodeproj
```
选择模拟器或真机运行。
